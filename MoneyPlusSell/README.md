# MoneyPlusSell  
  
  
経済システムプラグイン  
  
  
  
### 説明  
  
OP権限を持っている方のみが建てることができる看板式売却システムです。
  
  
#### 看板の書き方
  
  1行目: mpsell(configにて変更可能)  
  2行目: 売る値段(単位は不要)  
  3行目: ItemID:Meta  
  4行目: 個数(単位は不要)  
  
  
※一行目のtextはconfigで設定可能ですが、  
内容がほかのプラグインと被らないようにお気を付けください。  
(競合して予期せぬ動作をする場合があります。)
